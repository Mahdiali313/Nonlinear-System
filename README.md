# Attention !!!
dont use **sqrt** or **nthroot** for enter your equation. use fractions power like  **x^(1/3)**

هنگامی که کد از شما معادله رو میخواد ، اگر تابعتون دارای رادیکال هستش ، اونو به صورت توان کسری بنویسید.مثلا ایکس به توان یک سوم.


dont forget to use * between sin/cos/tan... and numbers.  **!! dont write 2sin(x)!!!**  correct form is **2*sin(x)**

حواستون هم باشه که بین اعداد و توابع مثلثاتی مثل سینوس و...  علامت ضرب * رو بزارید !
